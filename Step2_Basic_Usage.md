# Step 2: Claude Code 기본 사용법

이 문서에서는 Claude Code의 핵심 기능인 **파일 읽기/편집, 코드 생성, 터미널 명령 실행** 등 일상적으로 사용하는 기본 기능을 안내합니다.

## 1. 대화로 코드 작성하기

Claude Code는 자연어 지시만으로 코드를 생성합니다. 프로젝트 폴더에서 `claude`를 실행한 뒤, 원하는 작업을 설명하면 됩니다.

```
> Python으로 피보나치 수열을 계산하는 함수를 만들어줘
```

Claude는 코드를 생성하고, 파일로 저장할지 물어봅니다. 승인하면 즉시 파일이 생성됩니다.

## 2. 기존 코드 읽기 및 질문하기

프로젝트 내 파일에 대해 자유롭게 질문할 수 있습니다.

```
> README.md 파일을 읽고 요약해줘
> src/main.py에서 calculate 함수가 어떤 역할을 하는지 설명해줘
> 이 프로젝트의 전체 구조를 알려줘
```

Claude는 해당 파일을 직접 읽고 분석한 뒤 답변합니다.

## 3. 코드 수정 및 리팩토링

기존 코드를 수정하거나 개선하는 것도 대화로 가능합니다.

```
> app.js의 에러 처리를 try-catch로 감싸줘
> utils.py의 함수명을 snake_case로 바꿔줘
> 이 코드에서 중복된 부분을 리팩토링해줘
```

수정 전후의 diff(차이점)를 보여주며, 승인 후 적용됩니다.

## 4. 터미널 명령 실행

Claude Code는 필요에 따라 터미널 명령도 실행할 수 있습니다.

```
> npm install express를 실행해줘
> 현재 디렉토리의 파일 목록을 보여줘
> git status를 확인해줘
```

명령 실행 전에 사용자에게 승인을 요청하므로 안전합니다.

## 5. 권한 모드 (Permission Modes)

Claude Code는 실행 시 도구 사용 권한을 제어하는 모드를 제공합니다.

| 모드 | 설명 |
|------|------|
| **기본 모드** | 모든 도구 사용 전 승인 요청 |
| `--allowedTools` | 특정 도구만 자동 승인 허용 |
| `--dangerously-skip-permissions` | 모든 권한 검사 생략 (CI/자동화용, 주의 필요) |

```bash
# 예시: Bash와 파일 편집만 자동 허용
claude --allowedTools Bash,Edit
```

## 6. 이미지 및 파일 첨부

Claude Code는 멀티모달을 지원합니다. 이미지 파일 경로를 알려주면 시각적으로 분석할 수 있습니다.

```
> screenshot.png 파일을 보고 UI에서 문제점을 찾아줘
> diagram.png를 분석해서 설명해줘
```

## 7. 대화 중 유용한 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl + C` | 현재 응답 중단 |
| `Ctrl + D` | Claude Code 종료 |
| `↑` / `↓` | 이전/다음 입력 히스토리 탐색 |
| `Shift + Tab` | 여러 줄 입력 모드 전환 |

---
> 💡 **Tip:** 질문이나 지시는 구체적일수록 좋습니다. "이 코드 고쳐줘"보다 "이 함수에서 null 체크를 추가해줘"처럼 명확하게 요청하면 더 정확한 결과를 얻을 수 있습니다.
