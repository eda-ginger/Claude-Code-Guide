# Claude-Code-Guide

이 프로젝트는 **Claude Code** (AI 코딩 어시스턴트 및 Antigravity 에이전트)의 다양한 기능을 사용해 보고 테스트하기 위한 실습 및 연습용(Scratchpad) 저장소입니다.

## 목적 (Objective)
- AI 코딩 어시스턴트의 프롬프트 및 도구(Tools) 활용 실습
- 에이전트 기반 코드 작성, 리팩토링, 디버깅 최적의 협업 패턴 발굴
- 안전한 샌드박스 환경에서의 다양한 파이프라인 및 워크플로(workflows) 실험
