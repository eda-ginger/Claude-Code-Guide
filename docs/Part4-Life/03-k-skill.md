# k-skill — 한국 생활 밀착형 AI 스킬 모음

> Claude Code에서 한국 생활 서비스를 바로 사용할 수 있는 25개 이상의 스킬 패키지

---

## 개요

[k-skill](https://github.com/NomaDamas/k-skill)은 한국 사용자를 위한 AI 에이전트 스킬 모음입니다. Claude Code, Codex, OpenCode 등 코딩 에이전트에서 사용할 수 있으며, 교통, 쇼핑, 금융, 스포츠, 생활 정보 등 다양한 한국 서비스를 자연어로 제어할 수 있습니다.

---

## 설치

> 자세한 설치 방법은 [공식 설치 가이드](https://github.com/NomaDamas/k-skill/blob/main/docs/install.md)를 참고하세요.

### 1단계: 스킬 설치

```bash
npx --yes skills add NomaDamas/k-skill --all -g
```

26개 스킬이 `~/.agents/skills/`에 설치되며, Claude Code에 자동으로 symlink 됩니다.

### 2단계: 의존성 설치

```bash
# Node.js 의존성
npm install -g @ohah/hwpjs kbo-game kleague-results lck-analytics \
  toss-securities k-lotto used-car-price-search \
  cheap-gas-nearby korean-law-mcp daiso

# Python 의존성 (SRT/KTX 예매용)
python3 -m pip install SRTrain korail2 pycryptodome
```

### 3단계: NODE_PATH 영구 설정

Node.js 글로벌 패키지를 스킬에서 찾을 수 있도록 환경 변수를 설정합니다:

```bash
echo 'export NODE_PATH="$(npm root -g)"' >> ~/.bashrc
source ~/.bashrc
```

### 4단계: 설치 확인

```bash
# 스킬 디렉토리 확인
ls ~/.claude/skills/ | grep -c .   # Claude Code에 연결된 스킬 수

# Node 의존성 확인
node -e "require('@ohah/hwpjs'); console.log('hwpjs OK')"

# Python 의존성 확인
python3 -c "import SRTrain; print('SRTrain OK')"
```

### 5단계: 셋업 유틸리티 (선택)

인증이 필요한 스킬(SRT, KTX, 토스증권 등)을 사용하려면 Claude Code에서:

```
k-skill-setup 스킬을 사용해서 공통 설정을 진행해줘
```

> 인증 없이 사용 가능한 스킬이 대부분이므로, 먼저 인증 불필요 스킬부터 사용해볼 수 있습니다.

### 트러블슈팅

| 증상 | 해결 |
|------|------|
| `npm error 404` | 일부 패키지명이 변경될 수 있음. [공식 설치 가이드](https://github.com/NomaDamas/k-skill/blob/main/docs/install.md) 최신 버전 확인 |
| 스킬이 Claude Code에서 안 보임 | `~/.claude/skills/`에 symlink가 있는지 확인. 없으면 `npx skills add` 재실행 |
| `NODE_PATH` 관련 에러 | `echo $NODE_PATH`로 경로 확인. 비어있으면 3단계 재실행 |
| Python import 에러 | `python3 -m pip install --upgrade SRTrain korail2 pycryptodome` |

## 스킬 카테고리 및 동작 상태

> **상태 범례**: ✅ 테스트 완료 · ⚠️ 부분 동작/환경 제약 · 🔑 인증/API 키 필요
>
> k-skill은 초기 프로젝트로, 일부 스킬은 npm 패키지나 실행 스크립트가 아직 배포되지 않았습니다. **설치 직후 바로 사용 가능한 스킬은 ✅ 표시된 것들**입니다.

### ✅ 설치 후 바로 사용 가능 (인증 불필요, 테스트 완료)

| 스킬 | 설명 | 비고 |
|------|------|------|
| 로또 당첨 확인 | 회차별 당첨 번호 조회 | `k-lotto` npm 패키지 |
| KBO 야구 | 경기 결과 및 순위 | `kbo-game` npm 패키지 |
| K리그 | 축구 순위 및 결과 | `kleague-results` npm 패키지 |
| LCK e스포츠 | 리그 분석 및 결과 | `lck-analytics` npm 패키지 |
| 미세먼지 | 지역별 대기질 조회 | k-skill-proxy 경유 |
| 한강 수위 | 실시간 수위/유량 | k-skill-proxy 경유 |
| 서울 지하철 | 실시간 도착 정보 | k-skill-proxy 경유 |
| 맞춤법 검사 | 한국어 맞춤법 교정 | 스킬 내장 Python 스크립트 |
| 조선왕조실록 | 실록 검색 + 국역/원문 | 스킬 내장 Python 스크립트 |
| 우편번호 조회 | 주소별 우편번호 검색 | 우체국 공식 검색 |
| 중고차 시세 | SK렌터카 인수가/월렌트 조회 | 공개 inventory snapshot |
| 택배 조회 | CJ대한통운/우체국 배송 추적 | 공식 조회 표면, 송장번호 필요 |

### 🚫 현재 미지원 또는 추후 지원 예정

| 스킬 | 설명 | 사유 |
|------|------|------|
| 다이소 재고 조회 | 매장별 재고 확인 | npm 패키지 미출시. 재고 API 미동작 |
| 올리브영 재고 조회 | 매장별 재고 확인 | 다이소와 동일 패키지 사용. 미동작 |
| 주변 맛집 | 블루리본 기반 검색 | 프록시 세션 키 미설정으로 조회 불가 |
| 주변 술집 | 카카오맵 기반 검색 | 동작 미확인 |
| 쿠팡 상품 검색 | 상품 조회 및 비교 | MCP 서버 API 호출 제한 |
| HWP 변환 | 한글 문서 변환 | `@ohah/hwpjs` 설치됨. HWP 파일로 테스트 필요 |

### 🔑 인증 또는 API 키 필요

| 스킬 | 설명 | 필요 조건 |
|------|------|-----------|
| SRT 예매 | SRT 열차 조회/예매 | SRT 계정 필요. 테스트 미완 |
| KTX 예매 | KTX 열차 조회/예매 | 코레일 anti-bot 차단(MACRO ERROR). 우회 helper가 SKILL.md에 명시돼 있으나 실제 스크립트 미구현 (2026-04 기준) |
| 토스증권 | 포트폴리오/시세 조회 | macOS + Homebrew + `tossctl` 로그인 |
| 카카오톡 macOS CLI | 메시지 전송 | macOS + 카카오톡 앱 |
| 부동산 실거래가 | 아파트/주택 거래 내역 | `DATA_GO_KR_API_KEY` |
| 최저가 주유소 | 주변 주유소 가격 비교 | `OPINET_API_KEY` |
| 법령 검색 | 한국 법률/법령 조회 | `LAW_OC` API 키 ([발급](https://open.law.go.kr)) |

## 사용 예시 (테스트 완료)

설치 후 Claude Code에서 자연어로 바로 사용할 수 있습니다. 아래는 실제 테스트를 거친 예시입니다.

### 로또 당첨번호 조회

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `이번주 로또 당첨번호 알려줘` | 최신 회차 번호 6개 + 보너스 번호 |
| `1218회 로또 결과 알려줘` | 해당 회차 번호, 당첨금, 당첨자 수 |
| `내 번호 3, 10, 14, 15, 23, 24 몇 등인지 봐줘` | 번호 대조 후 일치 개수 및 등수 |

> 결과에 추첨일, 등수별 당첨자 수, 1인당 당첨금, 자동/수동/반자동 비율까지 포함됩니다.

### KBO 경기 결과

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `어제 KBO 경기 결과 알려줘` | 전 경기 스코어 + 승리투수 요약 테이블 |
| `2026-04-05 한화 경기 스코어` | 해당 날짜 한화 경기만 필터링 |
| `이번주 KBO 일정 정리해줘` | 날짜별 경기 일정 테이블 |

> 비시즌이나 경기 없는 날은 빈 결과가 나옵니다. 홈팀/원정팀, 구장, 선발투수 정보도 포함됩니다.

### 미세먼지 조회

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `종로구 미세먼지 알려줘` | PM10/PM2.5 수치 + 등급 (좋음/보통/나쁨) |
| `강남구 공기질 어때?` | 측정소, 조회 시각, 통합대기등급 |

> 지역명은 `강남구`, `종로구` 같은 행정구역명이 가장 정확합니다. 일부 시간대에 측정값이 없을 수 있습니다.

### 맞춤법 검사

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `"안녕하세여 저는 회사에 다닙니따" 맞춤법 검사해줘` | 교정 결과 + 원문/교정안/이유 목록 |
| `README.md 한국어 맞춤법 검수해줘` | 파일 내 오류 항목별 교정 제안 |

> 국립국어원 계열 규칙 기반. 원문 → 교정안 → 이유 형태로 결과가 정리됩니다.

### 서울 지하철 실시간 도착

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `강남역 지하철 도착 정보 알려줘` | 호선별 상/하행 도착 예정 시간 테이블 |
| `서울역 1호선 언제 와?` | 해당 노선 도착 메시지 |

> 실시간 데이터로 몇 초 단위 변동 가능. 조회 시점이 함께 표시됩니다.

### 한강 수위

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `한강대교 수위 알려줘` | 현재 수위(m), 유량(m³/s), 기준 수위 |
| `잠수교 유량 어때?` | 관측소명, 관측 시각, 수위/유량 |

> 10분 단위 갱신. 관심/주의/경보/심각 기준 수위와 비교해 줍니다.

### 조선왕조실록 검색

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `이순신 관련 조선왕조실록 검색해줘` | 총 결과 수 + 왕별 분류 + 기사 제목/국역/원문/링크 |
| `세종 때 측우기 기록 찾아줘` | 왕 필터 적용된 검색 결과 |

> 총 717건(이순신 기준) 중 상위 결과가 국역+원문+출처 링크와 함께 정리됩니다.

### K리그 경기 결과

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `최근 K리그 경기 결과 알려줘` | 경기 스코어 + 현재 순위 테이블 |
| `2026-04-05 FC서울 결과랑 순위` | 해당 팀 경기 필터 + 순위 |

> K리그1/K리그2 구분 가능. 관중 수, 중계 채널 정보도 포함됩니다.

### LCK e스포츠

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `LCK 최근 경기 결과 알려줘` | 매치별 세트 스코어 + 현재 순위 |
| `T1 경기 결과 보여줘` | 팀 alias 정규화 후 필터링 |

> BO3 세트 스코어, 블루/레드 사이드, VOD 유무까지 포함됩니다.

### 우편번호 조회

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `테헤란로 152 우편번호 알려줘` | 우편번호 + 주소 + 건물명 목록 |
| `세종대로 209 우편번호 찾아줘` | 후보 주소 중 선택 가능 |

> 우체국 공식 검색 기반. 같은 주소에 여러 건물이 있으면 후보 목록이 표시됩니다.

### 중고차 시세 조회

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `아반떼 중고차 시세 알려줘` | SK렌터카 기준 인수가 + 월렌트료 + 연식/주행거리 |
| `캐스퍼 인수가 얼마야?` | 해당 차종 재고 목록 + 가격 범위 |

> SK렌터카 다이렉트 타고BUY 공개 데이터 기반. 실시간 재고는 변동될 수 있습니다.

### 택배 조회

| 질문 방식 | 예상 결과 |
|-----------|-----------|
| `CJ대한통운 송장 1234567890 조회해줘` | 현재 상태 + 최근 이벤트 3건 (시각/위치/상태) |
| `우체국 택배 1234567890123 어디야?` | 배송 상태 + 이벤트 이력 |

> CJ대한통운(10/12자리)과 우체국(13자리) 지원. 송장번호가 필요합니다.

### 기타 (인증 필요 또는 미테스트)

```
서울역에서 부산까지 SRT 열차 조회해줘    # SRT 계정 필요
개인정보보호법 검색해줘                  # LAW_OC API 키 필요
쿠팡에서 에어팟 검색해줘                 # MCP 연결은 되나 API 제한 빈번
```

## 인증 설정

대부분의 스킬은 인증 없이 바로 사용 가능합니다. 인증이 필요한 스킬은 `k-skill-setup`을 통해 설정합니다:

- **SRT/KTX**: 각 서비스 계정 정보
- **토스증권**: 토스 인증 정보
- **카카오톡**: macOS 카카오톡 앱 연동
- **부동산/주유소**: 공공데이터포털 API 키 (`DATA_GO_KR_API_KEY`, `OPINET_API_KEY`)
